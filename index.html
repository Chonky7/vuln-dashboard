Cisco PSIRT Vulnerability Tools
Official Cisco Security Advisory data — model & version aware
Why use this instead of NVD for Cisco?
NVD indexes CVEs by software name (FTD, FXOS, IOS XE…), not hardware model numbers. Searching for "Firepower 4112" in NVD returns nothing because CVE descriptions say "Cisco Firepower Threat Defense Software" or "Cisco FXOS".

The Cisco Software Checker and PSIRT openVuln API know that the Firepower 4112 runs FTD + FXOS and map model → software → advisory automatically. They are the authoritative source for Cisco vulnerability data.
